1- Diferenças entre DevOps e SRE

- DevOps é uma função que faz a junção do desenvolvimento e operações, focando em colaboração, automação e entrega contínua (CI/CD).
- SRE é a aplicação da engenharia de software para resolver problemas de operações, com foco em confiabilidade e error budgets.

 DevOps é o "o quê", SRE é um "como" fazer isso na prática.
